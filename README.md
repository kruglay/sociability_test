##Тест на общительность

Учебная программа. 

Тест на общительность. 

Чтобы узнать насколько вы общительны вам нужно ответить на 16 вопросов, где ответ a - 2 очка, b - 0 очка, c - 1 очка

####Установка

Скопировать программу на компьютер

####Запуск

В строке консоли набрать
```
ruby main.rb
```
